## 🎨 구독 티어 설계 (상세)

### 라이트 티어 (3,000원/월)

```yaml
목표: 진입 장벽 낮추기
타겟: 신규 팬, 여러 아이돌 동시 구독

혜택:
  - 전용 피드 열람 (주 2~3회 업데이트)
  - 월 1회 디지털 포토카드
  - 구독자 전용 뱃지
  - 댓글 우선 답변 (50% 확률)

크리에이터 부담:
  - 주 2~3회 전용 피드 작성
  - 월 1회 포토카드 제작
```

### 프리미엄 티어 (10,000원/월) ⭐ 메인

```yaml
목표: 메인 수익원
타겟: 진성 팬 (70% 비중)

혜택:
  - 라이트 혜택 전체
  - 버블 메시지 무제한 (실시간 소통)
  - 월 3회 디지털 포토카드 (고화질)
  - 주 1회 비하인드 영상/사진
  - 생일 축하 영상 메시지
  - 댓글 우선 답변 (100%)

크리에이터 부담:
  - 주 5~7회 버블 메시지 (1일 1회 목표)
  - 주 1회 비하인드 컨텐츠
  - 구독자 생일에 영상 제작
```

### VIP 티어 (30,000원/월)

```yaml
목표: 열성 팬 보상
타겟: 소수 핵심 팬 (10명~50명)

혜택:
  - 프리미엄 혜택 전체
  - 월 1회 1:1 영상통화 (5분)
  - 분기별 친필 편지
  - 이벤트 우선 예약 (최우선)
  - 연간 오프라인 팬미팅 무료 초대
  - 한정판 포토카드 (실물 우편 발송)

크리에이터 부담:
  - 월 1회 영상통화 진행
  - 분기별 편지 작성 (3개월마다)
  - VIP 전용 이벤트 기획
```

## 💸 수수료 정책

### 플랫폼 수수료

```
정기 구독:
  플랫폼 수수료: 20%
  결제 수수료: 5%
  크리에이터 수령: 75%

  예) 10,000원 구독
    → 플랫폼: 2,000원
    → 결제사: 500원
    → 크리에이터: 7,500원

일회성 후원:
  플랫폼 수수료: 15%
  결제 수수료: 5%
  크리에이터 수령: 80%

  예) 10,000원 후원
    → 플랫폼: 1,500원
    → 결제사: 500원
    → 크리에이터: 8,000원
```

### 수수료 차등제 (활동 독려)

```
🏆 활동 점수별 수수료 할인

전설 크리에이터 (500점+): -5% (수수료 15%)
마스터 크리에이터 (300~499점): -3% (수수료 17%)
전문가 크리에이터 (150~299점): -2% (수수료 18%)
활발한 크리에이터 (50~149점): -1% (수수료 19%)
신인 크리에이터 (~49점): 정상 (수수료 20%)

→ 컨텐츠를 많이 생산할수록 수수료 감소!
```
